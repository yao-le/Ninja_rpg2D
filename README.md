### 忍者RPG小游戏


- A：向左
- D：向右
- W：向上
- S: 向下
- R: 复活/重新开始
- E：和可交互NPC互动
- 鼠标选中敌人后按空格键攻击
- 鼠标左键点击被打倒的敌人，拾取装备
